An HTTP load balancer that doesn't require a PhD to operate.

lithe is a single-binary Go reverse proxy and load balancer. One config file, hot reload on SIGHUP, zero external dependencies.

lithe.yaml 
# lithe.yaml — minimal config
listen: "0.0.0.0:443"

tls:
  provider: acme       # auto-renews
  email: "ops@corp.io"

upstreams:
  - name: api
    backends:
      - 10.0.0.10:8080
      - 10.0.0.11:8080
      - 10.0.0.12:8080
    strategy: least-conn
    health_check:
      path: /healthz
      interval: 5s

routes:
  - match: "/api/*"
    upstream: api
